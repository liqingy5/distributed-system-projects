# Group chat Server/Client

Group chat system utilizing Raft Concesus Algorithm 
## Prerequisites

- Python 3.11.1
- pip 22.3.1

### Set up virtual environment

```Bash
python -m venv env
```

```Bash
source env/bin/activate
```

### Install dependencies

```Bash
pip install -r requirements.txt.
```

## Generate gRPC code

```Bash
source generate_proto.sh
```
